Linus Torvalds

One of the great inavators of the computer age

Linus Benedict Torvalds is a Finnish-American software engineer who is the creator and principal developer of the Linux kernel, which is the kernel for the Linux operating systems and other operating systems such as Android and Chrome OS. He also created the distributed version control system Git.

Early Years

Linus, Peanuts or Chemist?

Torvalds was born in Helsinki, Finland, on 28 December 1969.Torvalds was named after Linus Pauling, the Nobel Prize-winning American chemist, although Torvalds is quoted as saying, "I think I was named equally for Linus the Peanuts cartoon character", noting that this makes him half "Nobel Prize-winning chemist" and half "blanket-carrying cartoon character".

His interest in computers began with a Commodore VIC-20, at the age of 11 in 1981, initially programming in BASIC, but later by directly accessing the 6502 CPU in machine code (coding with 1s and 0s). After the VIC-20 he purchased a Sinclair QL, which he modified extensively, especially its operating system. "Because it was so hard to get software for it in Finland, Linus wrote his own assembler and editor.

Linux

A Free Open-source Operating System

The first prototypes of Linux were publicly released in 1991.Version 1.0 was released on 14 March 1994.The first prototypes of Linux were publicly released in 1991.Version 1.0 was released on 14 March 1994.

Torvalds first encountered the GNU Project in 1991, after another Swedish-speaking computer science student, Lars Wirzenius, took him to the University of Technology to listen to free software guru Richard Stallman's speech. Torvalds used Stallman's GNU General Public License version 2 for his Linux kernel.

Why Linux Rocks

1. Open Source

It's code is free and readily available for anyone to fully customize and distribute for any purpose.

2. Security

Every program in Linux needs authorization from the admin via password, making linux less vunerable to malware than alternatives.

3. Large Community Support

Linux has a large and active user base that is constantly updating and improving various linux distributions.

4. Stable and Reliable

The Windows operating system needs to repoot afting installing certain applications/softwarare or when performance lags. Linux can run at high performance without requiring many/frequent reboots

5. Privacy

linux collects minimal data from user unlike other operating systems.
